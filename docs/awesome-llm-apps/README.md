# Awesome LLM Apps

A curated list of awesome Large Language Model applications, tools, and resources.

## AI Tools

- [LangChain](https://github.com/langchain-ai/langchain) - Framework for developing applications powered by language models
- [LlamaIndex](https://github.com/run-llama/llama_index) - Data framework for LLM applications
- [AutoGPT](https://github.com/Significant-Gravitas/AutoGPT) - An experimental open-source attempt to make GPT-4 fully autonomous

## Chatbots

- [Transformer Chatbot](https://github.com/bentrevett/pytorch-seq2seq) - A PyTorch implementation of a sequence-to-sequence model for chatbot
- [ChatGPT Clone](https://github.com/atlassian/react-spring-boot-chatbot) - ChatGPT clone built with React and Spring Boot

## Data Analysis

- [Pandas AI](https://github.com/gventuri/pandas-ai) - Pandas AI is a library that integrates LLMs into pandas
- [Code Interpreter](https://github.com/shroominic/codeinterpreter-api) - Code interpreter API for LLMs

## Agent Frameworks

- [CrewAI](https://github.com/joaomdmoura/crewAI) - Framework for orchestrating role-playing autonomous AI agents
- [AutoGen](https://github.com/microsoft/autogen) - Enabling the development of LLM applications using multiple agents
